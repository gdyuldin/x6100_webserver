Welcome to X6100 alternative firmware configuration editor.

Bands editor

Allows to add/change/remove bands. Active column is to bind band to up/down buttons.

Digital modes editor

Allows to add/change/remove frequencies for FT8/FT4

File browser

Allows to download files from the DATA partition of the SD card (ADI file, screenshoots, recordings, etc).

Note: there is a small probability, that params.db might be changed during downloading and downloaded version might be corrupted. This will not affect params.db on the radio. You can check downloaded file with opening it using any SQLite3 tool.
